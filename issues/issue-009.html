GPT-5.6 正式公开上线：Sol/Terra/Luna 三档，ultra 并行智能体

OpenAI 于 7 月 9 日正式向全球开放 GPT-5.6 系列，分旗舰 Sol（$5/$30 每百万 token）、均衡 Terra（$2.50/$15）、轻量 Luna（$1/$6）三档。Sol 在编码基准（Terminal-Bench 91.9%、ExploitBench 73.5%）超越 Fable 5，且仅用约 ½ token、⅓ 成本。新增 ultra 模式协调 4 路并行智能体处理复杂任务，同步推出系统卡（System Card）与生物安全漏洞赏金计划。

SpaceXAI 联同 Cursor 发布 Grok 4.5，正面叫板 GPT-5.6

马斯克旗下 SpaceXAI 于 7 月 9 日公开上线 Grok 4.5，由 1.5 万亿参数 MoE 架构驱动、trillions of Cursor 交互 token 联合训练，运行于 Colossus 集群（22 万+ GPU）。Terminal-Bench 2.1 达 83.3%，自称 Opus 级但更快更便宜——Cursor 内 API $2/$6 每百万 token（快速通道 $4/$18）。在 GPT-5.6 同日上线、Fable 5 受限于出口管制的背景下，三强定价竞争正式开打。

ChatGPT Work + GPT-Live：OpenAI 发布工作智能体与全双工语音

OpenAI 同期发布两大新产品——ChatGPT Work 是运行于桌面与 Web 的工作智能体，可跨 Slack、Teams、Google Drive 等应用自主执行多步工作流、定时调度任务（Scheduled Tasks），并将在 ChatGPT 内创建交互式 Sites；Codex 并入桌面应用。GPT-Live 则实现全双工语音架构（可同时听与说），智能体可暂停、打断、后台调用 GPT-5.5 进行搜索推理，GPT-Live-1 与 mini 版即日起全球上线。用户量超 1.5 亿人/周。

SWE-1.7：Cognition 基于 Kimi K2.7 的极低成本前沿编码模型
Cognition 7 月 8 日发布 SWE-1.7，架构基于 Moonshot AI 的 Kimi K2.7 经 Cognition 强化学习管道深度训练。在 Cerebras 上推理速度达 1000 token/秒，FrontierCode 1.1 达 42.3%（远超 SWE-1.6 的 9.4%），以远低于 Opus 4.8 和 GPT-5.5 的成本逼近前沿水准，证明 RL 可进一步推高已后训练模型的性能天花板。

模型
Muse Spark 1.1：Meta 发布智能体编码模型，首次对 AI 能力收费
Meta 7 月 9 日发布 Muse Spark 1.1，是其迄今最强的智能体编码模型，支持 100 万 token 上下文、多模态理解（文本/图像/视频）及多智能体协调。扎克伯格时隔近三年重返 X 宣布此事。模型向开发者开放预览并公布收费方案，标志 Meta 正式进入 AI 付费竞争市场，直接对标 OpenAI 和 Anthropic。

评测
OpenAI 审计发现 SWE-Bench Pro 约 30% 任务存在缺陷，撤回推荐
OpenAI 7 月 8 日发布对 SWE-Bench Pro 的深度审计，通过数据质量分析管道与 5 名资深工程师标注，发现约 30% 任务破损——问题包括测试过严、提示欠描述、低覆盖测试及误导性提示。OpenAI 正式撤回此前对 SWE-Bench Pro 的推荐，呼吁社区构建由专业开发者从头设计的新基准。此次审计本身使用 Codex 智能体辅助完成。

安全
工信部对 Claude Code 发出国家级安全后门风险提示
7 月 8 日，工业和信息化部网络安全威胁和漏洞信息共享平台（NVDB）发布公告，指 AI 编程工具 Claude Code 自 2.1.91 版本起内置监控机制，未经用户同意向远程服务器回传地域、身份标识等敏感信息，危害严重。工信部建议相关用户立即卸载受影响版本（2.1.91–2.1.196）。此前阿里已宣布 7 月 10 日起内部禁用 Claude Code，此次国家级警告将进一步影响该工具在中国的使用前景。

前沿模型进入性价比军备竞赛：Sol vs Grok 4.5 vs Luna 横向定价
