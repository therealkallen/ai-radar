Bonsai 27B：首个 27B 级手机上运行的多模态模型，1-bit 仅 3.9GB

PrismML（Caltech 团队，Khosla/Cerberus/Google/Samsung 投资）7 月 14 日发布 Bonsai 27B，基于 Qwen3.6 27B，是全球首个可在手机本地运行的 27B 级多模态模型。Ternary 版（5.9GB，1.71 有效比特/权重）保留全精度 95% 基准分；1-bit 版（3.9GB，1.125 比特/权重）保留 90%，可在 iPhone 17 Pro 上运行。数学和编码能力几无损失（91.7%–93.4%），工具调用保留率 66%–74%。Apache 2.0 开源。HN 535 分——「智能密度」范式正在将前沿能力推向边缘设备。

压缩
Bonsai 27B：智能密度范式将 27B 前沿能力压缩到手机内存
PrismML 的 Bonsai 27B 通过全端到端三值/二值化（覆盖 embedding、attention、MLP 和 LM head，无高精度逃生通道），在 262K token 上下文、支持视觉多模态 + 推测解码的条件下，将 27B 模型从 ~54GB（FP16）压缩至 5.9GB/3.9GB。1-bit 版智能密度达 0.53/GB——是全精度基线的 10x+，最佳常规低比特方案的 2.7x。在 M5 Max 上达 58–87 tok/s，端侧 Agent 闭环的成本已降为零。

美团 LongCat-2.0：首个在五万卡国产算力集群训练的 1.6T 参数大模型
美团 7 月 14 日正式发布 LongCat-2.0，1.6T 总参数（~48B 稀疏激活），百万 token 上下文，专为 Agentic Coding 设计。关键突破是首次在 50,000 张国产 GPU 集群上完成万亿参数级训练，并同步开源国产卡推理代码。配套发布 WBench（首个交互式视频世界模型评测基准）和 VitaBench 2.0（长期动态智能体基准），形成「模型+评测」的完整开源栈。

DRACO 公开测试：国产四模型集成方案以 1/14 成本匹敌 GPT-5.6-sol
DRACO Brave Search 组公开评测中，GPT-5.6-sol 入榜得分 63.99，单任务成本 $1.71；由 DeepSeek、GLM、Kimi、Qwen 四模型并行提案再聚合的 OpenSquilla 0.5.0 方案得分 64.09，成本仅 $0.12——质量持平，成本为 GPT-5.6-sol 的 1/14、Fable 5 的 1/10。该结果指向一个实用路径：国产模型通过集成路由策略可在特定任务上实现与前沿闭源模型相当的性价比。

OpenAI 发布「智能体时代的 AI 投资管理」框架：五个步骤从实验走向规模化
OpenAI 7 月 14 日发布企业 AI 投资管理指南，提出五步框架：① 建立用量与支出可视化（按用户/产品/模型）；② 以「每次可接受成果的成本」而非 token 单价评估模型效率；③ 在规模化前建立治理层（插件权限、操作审批、容量分配）；④ 以组合思维投资 AI：日常生产力 → 可重复工作流 → 战略性私有数据工作流；⑤ 用 Guaranteed Capacity / Scale Tier / Batch API 等分层匹配产能与需求。文章同时透露 GPT-5.6 相比 GPT-5.4 输出 token 减少 54%、任务时间缩短 57%。
